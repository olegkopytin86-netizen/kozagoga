# 🐐 Козагога — Маркетплейс цифровых товаров

Современный маркетплейс для покупки цифровых товаров: пополнение игр и кошельков, подарочные карты, игровые ключи, аккаунты и многое другое.

## 🚀 Технологии

- **React 19** + **TypeScript** — современный UI
- **Vite 7** — быстрая сборка
- **Tailwind CSS v4** — стилизация
- **Framer Motion** — анимации
- **React Router v7** — маршрутизация
- **Lork SDK** — бэкенд и база данных
- **shadcn/ui** — компоненты

## 📦 Установка и запуск

```bash
# Установка зависимостей
pnpm install

# Запуск в режиме разработки
pnpm dev

# Сборка для продакшена
pnpm build

# Предпросмотр собранного проекта
pnpm preview
```

## 🏗️ Структура проекта

```
src/
├── components/     # UI компоненты
│   └── ui/        # shadcn/ui компоненты
├── contexts/      # React контексты (Auth, Cart)
├── lib/           # Утилиты
├── pages/         # Страницы приложения
├── types/         # TypeScript типы
├── App.tsx        # Главный компонент
├── main.tsx       # Точка входа
├── router.tsx     # Маршрутизация
└── index.css      # Глобальные стили
```

## 📄 Страницы

- `/` — Главная
- `/catalog` — Каталог товаров
- `/product/:id` — Детальная страница товара
- `/cart` — Корзина
- `/checkout` — Оформление заказа
- `/orders` — Мои заказы
- `/order/:id` — Детали заказа
- `/profile` — Профиль пользователя
- `/login` — Вход
- `/register` — Регистрация
- `/dashboard` — Панель управления
- `/admin` — Админ-панель
- `/faq` — Часто задаваемые вопросы
- `/contacts` — Контакты
- `/about` — О нас
- `/search` — Результаты поиска
